#Catálogo de Pokemones

Aplicación web desarrollada con Next.js que muestra un catálogo de Pokémon consumiendo la PokeAPI.  
La página permite navegar entre diferentes Pokémon, ver su nombre e imagen, y acceder a una ruta dinámica con el detalle de cada uno.


Funcionalidades

- Listado de Pokémon con nombre e imagen
- Menú de navegación dinámico
- Rutas dinámicas por Pokémon (`/pokemon/[name]`)
- Consumo de la PokeAPI
- Uso del componente `next/image` para optimización de imágenes
- Estilos con Tailwind CSS
- Renderizado del lado del servidor (Server Components)
